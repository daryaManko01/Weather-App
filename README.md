# Weather App

A simple weather application built with vanilla JavaScript using the OpenWeather API.

## Features
- Search weather by city name
- Current temperature display
- Current date
- Hourly forecast (every 3 hours)
- Weather icons

## Technologies
- HTML
- CSS
- JavaScript
- OpenWeather API

## API
- Current weather: https://api.openweathermap.org/data/2.5/weather
- Forecast: https://api.openweathermap.org/data/2.5/forecast

## Features
- Auto-load default city
- Update data on button click
- Formatted hourly forecast with icons
